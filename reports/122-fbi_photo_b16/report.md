# #122 FBI Photo B16：又合起來了

![photo](photo.jpg)

時間戳：**12/31/99 18:20:41**。距 B15 19 秒。

B14、B15 都是兩個分開的物體。B16 不一樣。

畫面中央偏右，**一個拉長的暗色團塊**，左邊較大、右邊有個明顯較小的突起。看起來像是 B13 那種「主體 + 突起」的構造重現了。但 B13 是進入 zoom 模式的第一張，B16 是經過 B14、B15「明確分成兩個」之後的合併狀態。

**19 秒之內，兩個物體又合起來了。**

## 分離與合併的迴圈

把目前已知的形態變化串成一條時間線：

| 幀 | 時間 | 形態 |
|---|---|---|
| B13 | 18:19:54 | 主體 + 右側突起（模糊一體） |
| B14 | 18:20:08 | **兩個分開的圓塊** |
| B15 | 18:20:22 | 兩個分開（位置略左移） |
| **B16** | **18:20:41** | **又合起來了**，主體 + 小突起 |

47 秒之內，目標形態走了一個「**合 → 分 → 分 → 合**」的循環。

這個行為非常不像常規飛行器。飛機不會在空中拆解再重組。鳥群會分散與重新聚集，但時間尺度通常是分鐘級而不是 19 秒。**多個相同熱訊號的物體在 1 mil 範圍內動態地靠近、分開、再靠近**，符合 UAP 文獻中描述的「**morphing**」行為。

UAP morphing 的歷史紀錄包括：
- 1989 比利時三角形 UFO 浪潮：多份報告描述物體分裂成多個小光點再合併
- 2008 Stephenville TX：F-16 雷達追蹤的物體報告呈現形態變化
- 2023 AARO Historical Record Report：多份 Project Blue Book 案件描述「物體變形」

B16 提供了**軍方 IR sensor 捕捉到「合併狀態」**的可量化視覺證據。

## 不是 sensor 失焦造成的合併

要排除一個解釋：B16 的合併會不會只是 sensor 失焦造成的視覺假象？

兩個物體在 IR 中看起來「合在一起」可以是真的合併（物體靠近），也可以是 sensor 變得模糊（解析度下降）。要判別需要看背景銳利度。

對照 B16 與 B14 / B15 的背景（天空雜訊紋理的細節、左右兩側遮蔽長條的邊緣銳利度）**幾乎完全一致**。sensor 對焦沒有改變。

那 B16 的合併**不是失焦造成的**。是物體本身的相對位置變化。
