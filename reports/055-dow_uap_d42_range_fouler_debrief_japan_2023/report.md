## 為什麼 D42 顯示「Multiple UAP 相互超越 + 互動」是 D 系列罕見三體案

D 系列其他案件多為**單一 UAP 觀測**（D33, D35, D38, D27, D19）或**單一觀測時刻多 UAP**（D3 = 4 個分 3 波；D20 = 10-20 個於 FL600+ 散布；D23 = 2 個）。D42 是少數**連續觀測中物體數量動態變化**的案例：

1. **T0**：機組觀測 1 個物體 → 開始追蹤
2. **T1**：第 2 個物體出現，**形狀與大小相同但速度遠快於第一個**，超越第一個
3. **T2**：第 3 個物體出現，**3 個物體同時在畫面中相互移動**（moving amongst each other）

「**moving amongst each other**」（相互移動 / 互動）暗示物體之間存在**相對運動**，可能解讀為：

- **編隊飛行**：3 物體保持鬆散編隊但個別調整位置（如鳥群、無人機群）
- **獨立物體偶然在同一畫面**：3 物體無關聯，各自飛行
- **協同行動**：3 物體有目的地相對移動（如包圍、追擊、組編）

「**Apparent Propulsion checked**」進一步指出機組觀察到可見的**推進跡象**（如熱羽、尾跡、燈光），是 D 系列中少數明確標示物體有推進可見的案件。

## 1. 飛行員親述

機組描述（手寫填入「Description and Interaction」欄位）：

> "MGRS Grid of initial contact: 39RWL[REDACTED]
>
> Saw the initial object fly through the screen and started tracking it. Initial object was surpassed by another object of same size and shape but much higher speed. At one point during tracking the objects, there were three on the screen at the same time moving amongst each other."

> 「初始接觸 MGRS 座標：39R WL [遮蔽]
>
> 看到初始物體飛越畫面後開始追蹤。初始物體被另一個**形狀與大小相同但速度高出許多**的物體超越。追蹤過程中，曾有一個時刻畫面中同時出現**三個物體相互移動**。」

關鍵觀察：

- **同形不同速**：兩個（之後三個）物體 size + shape 相同，但 velocity 顯著差異
- **超越**（surpassed）：物體 #2 從後方追上並超過物體 #1
- **三體同框**：peak 時刻 3 個物體同時在 sensor 視野中
- **相互移動**（moving amongst each other）：物體間有相對運動，非單純並排或編隊

## 2. 「同形不同速」的物理意涵

物體 #1 與物體 #2 有相同 size + shape 但顯著不同 velocity。若假設兩者為同類物體：

| 物體 | size | shape | velocity |
|---|---|---|---|
| #1 | A | A | V₁ |
| #2 | A | A | V₂ >> V₁ |

可能解讀：

1. **同類載具的不同任務階段**：物體 #1 在低能耗模式，物體 #2 在高速截擊模式
2. **同類載具的能力差異**：物體 #2 性能更強（如不同版本）
3. **遠近錯覺**：物體 #2 可能更近，看起來更快（但 size 同 → 假設一致距離）
4. **同類但被不同方式驅動**

對照 [#041 D20 ESSA 2023-03-31](../041-dow_uap_d20_mission_report_syria_essa_march_2023/report.md)：F-16CM 觀測 10-20 個亮物中，**個別物體速度不同**。D42 與 D20 共構「多物體 + 速度差異」signature。

## 3. 地理：39R WL grid 對應波斯灣 / 阿曼灣 / 伊朗

座標「**39R WL**」依 MGRS 規範：

- **39R grid zone**：經度 48°E–54°E（**波斯灣中部 / 伊朗西南 / 沙烏地阿拉伯東部省份外海**）
- 緯度帶 R：24°N–32°N
- **WL square**：39R 內的 100 km 子方格（具體位置遮蔽）

39R WL 對應**波斯灣中部、伊朗布什爾 / 巴林 / 卡達周邊海域**，與標題「**Japan, 2023**」**完全不符**。標題錯誤可能來自：

1. **War.gov metadata 系統性錯誤**：對應 D27、D20、D4 等多份標題錯誤案件
2. **PDF 標題 vs. 上傳時的 metadata 操作錯誤**

482 ATKS（AFRC, Homestead ARB FL）是**美國本土的 AFRC 海軍預備部隊 MQ-9 部隊**，**通常派駐 USCENTCOM AOR（中東）**，**極少派駐日本**。對照 [#039 D18 巴格達 2022-12-01](../039-dow_uap_d18_mission_report_iraq_december_2022/report.md)，同為 482 ATKS 在中東執行的 UAP 觀測。**D42 + D18 構成 482 ATKS 跨年份觀測對**。

## 4. MGRS 39R + 18,000 ft + dusk = MQ-9 ISR 標準巡邏剖面

- **18,000 ft Altitude Constant Yes**：MQ-9 常規 ISR 巡航高度
- **Dusk**：黃昏時段，EO sensor 仍可用（IR + visible）但對比較差，是 UAP 觀測常見時段（如 D32 在 16:00Z 接近日落）
- **Radar Equipped: Other**：MQ-9 非典型雷達裝備（如 Lynx II SAR，與 F/A-18 ATFLIR + APG-79 不同）

對應 D 系列其他 dusk 案件：

- D32（2024-10-20 16:44Z, dusk）= AFSOC 12 SOS 觀測 5 次 light/glare
- D42（2020-08-31 dusk）= 482 ATKS 觀測 3 個同形物體
- D18（2022-12-01 dusk 前後）= 482 ATKS UAP/UAV 雙列

「**Dusk + IR-visible 雙模 sensor**」是 D 系列高頻觀測時段，可能因日落光照變化使 IR sensor 對冷物體對比增加，更易偵測。

## 5. 觀察

**(1) D42 + D18 構成 AFRC 482 ATKS 跨年觀測對**：D18 = 2022-12-01 Iraq, D42 = 2020-08-31 39R WL（波斯灣）。482 ATKS 是 Homestead ARB FL 的 AFRC Reaper 部隊，**至少累積 2 份 D 系列 UAP 通報**。

**(2) D 系列首次「同形不同速」描述**：物體 #2 surpassed 物體 #1 + 同 size/shape 但 much higher speed。對照 D20「10-20 個亮物中個別速度不同」、D33「90 度直角轉彎」，D42 提供**多物體 + 速度差異 + 互動**的 signature。

**(3) MDR 26-0028 與 D33-D38 不同批次**：D33 / D35 / D38 在 MDR 26-0019，D42 在 MDR 26-0028。**MDR 26-0028 釋出時間 2026-03-16，比 26-0019 晚 53 天**。意味 D42 解密晚於 D33/D35/D38。可能因 D42 涉及 AFRC（vs. USAF 主力部隊）需額外審查程序。

**(4) FOUO/PA applies 是 D 系列特殊註記**：FOUO（For Official Use Only）+ PA（Privacy Act applies）意味即使解密，**仍保留官方使用限制**（用於官方通信）+ **隱私法適用**（涉及個人識別資訊保護，即使 SPEAR 已去識別化）。對照其他 D 系列無此註記。

**(5) 「形狀: Other Shape」+「Apparent Propulsion: checked」**：表格 8 種形狀（Round / Square / Balloon-shaped / Wings/Airframe / Other Shape）中勾選「Other Shape」，意味物體形態不屬於常見類別。同時「Apparent Propulsion」勾選，意味機組看到可見推進跡象（不像普通氣球）。這是**形態異常 + 推進異常的雙重 signature**。
