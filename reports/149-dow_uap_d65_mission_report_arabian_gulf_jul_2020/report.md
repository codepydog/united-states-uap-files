# #074 DOW-UAP-D65：21 小時 3 個 UAP、3 種 IRGCN 艦艇、1 次 Bushehr 港口同框

![page-1](page-1.jpg)

2020-07-16 至 07-17，**482 ATKS** 的 MQ-9 Reaper 從 OKAS 起飛，**21 小時 17 分鐘**任務，又一次 NAVCENT 支援，建立伊朗船舶 pattern of life + 港口外活動特徵化。這次比 D62/D63/D64 早**將近 3 個月**，是 482 ATKS 部署到 NAVCENT 戰區後**較早**的 UAP 記錄。

21 小時內，3 個 UAP + 3 種 IRGCN 艦艇 + 1 次伊朗喊話 + Bushehr 港口同框觀測。**Weather: WAS NOT A FACTOR**（與 D63/D64 的 HAZE 形成對比）。

三筆觀測都用同樣的標準格式：「1X UNIDENTIFIED AERIAL PHENOMENON IVO [MGRS]」，沒有 shape / altitude / speed / heading 的細節。

UAP #1 與 UAP #2 間隔 **50 分鐘**，MQ-9 從 MGRS 39RXK 移到 39RUN，航向從 152M（東南）轉到 34M（東北）——大幅機動。

UAP #3 出現在 4 小時 25 分鐘後，MQ-9 在 39RUN68，航向 331M（西北西）——又一次大幅機動。

**UAP #3 的座標精度是 D 系列裡少見的**：`39RUN6234236874`，**10 位數**精度對應 1 公尺解析度。標準 MGRS 觀測通常只給 6 位數（100m）或 8 位數（10m），10 位數意味 MQ-9 sensor lock 到了一個非常精確的位置（也許是物體著地或精確 hover 的點）。

## 三種 IRGCN 艦艇：Naser、Safir Kish、Houdong

D63 觀測過 Naser WAP + Bushehr boatyard。D65 把這個敘事擴展到 3 種船型：

| 時刻 | 艦艇型 | 評估等級 | 位置 |
|---|---|---|---|
| 06:45Z | **Naser WAP** | PROB（probable） | 不適用 |
| 19:32Z | **Safir Kish** | PROB | IVO 39RUN6? 朝東 |
| 21:40Z | **Houdong + Naser** | POSS（possible） | Docked at **Bushehr** |

**Naser**：伊朗 IRGCN 自製快攻艇（small fast attack craft），長約 18 公尺，配 12.7mm 機槍 + 火箭發射器，主要用於荷莫茲海峽騷擾航運。

**Safir Kish**（可能寫作 Safir-class）：伊朗 IRGCN 改裝自漁船的小型砲艇，配備 12.7mm 機槍與輕型反艦飛彈（如 Nasr-1）。Kish 島在荷莫茲附近，IRGCN 在那裡有基地。

**Houdong**（霍紐東級）：中國製 Hudong-class 飛彈快艇，伊朗 IRGCN 在 1990 年代採購 10 艘，配備 C-802 反艦飛彈。這是 IRGCN **主力反艦平台**。

把這 3 種船在同一份 MISREP 都列出來，**D65 是 D 系列裡 IRGCN 艦艇 OOB（Order of Battle）覆蓋最完整的一份**。Bushehr 港口同時停 Houdong + Naser（飛彈快艇 + 快攻艇）的影像，是 NAVCENT 對 IRGCN 港口部署的關鍵情資。

## 為什麼 D65 沒有像 D63/D64 受霾影響

7 月中旬阿拉伯灣的氣候條件與 9-11 月不同。**夏季伊朗北部的 Shamal 季風**會把 Iraq + Iran 內陸沙塵吹進阿拉伯灣，但 7 月份還沒到 Shamal 高峰（通常是 8-9 月）。到 11 月秋季層雲與輻射霧也減少。

D65 比 D62/D63/D64 早 2-4 個月，所以 EO/IR 觀測能力**沒被霾打折**。這意味 D65 的 3 個 UAP 觀測**證據強度高於後三份**。

但很有趣的是，**即便能見度好，narrative 仍只記「1X UNIDENTIFIED AERIAL PHENOMENON」**，不寫 shape / altitude / speed。這支持以下推論之一：

1. MQ-9 sensor 操作員的記錄習慣就是這樣簡短，細節留給 DGS-1 FMV 影像判讀
2. 完整描述被 1.4(a) 遮蔽，narrative 只保留事件存在性
3. 三次 UAP 都是雷達 / sensor 偵測到無法判讀的物體，原本就沒有 visual 細節

這對應 [#072 D63](../147-dow_uap_d63_mission_report_strait_of_hormuz_oct_2020/report.md) 的 narrative 處理方式（也是「OBSERVED A UAP, SEE OBSERVATION LINE」），意味 **482 ATKS 的 MISREP 寫法在 2020 年下半年沒有改變**。
