# #108 FBI Photo A2：地面變得清楚了

![photo](photo.png)

A1 那張照片給人的第一印象是一片均勻的灰。除了中央的瞄準十字、邊邊那十二道黑色長條，其他什麼都看不到。背景看起來像高空中的天空、或者是離得太遠以致細節糊掉的某個東西。

A2 不一樣。

打開檔案的瞬間就能看出來：sensor 不再對著空無一物的某個方向，而是**俯視地面**。畫面下半部能辨認出乾燥地表的紋路，土色基底上散布著一些較深的小塊狀物。可能是沙漠中的灌木叢、可能是岩石、也可能是斷續出現的低矮植被。地表的紋理告訴我們兩件事：第一，這架載具當時飛在能看見地面細節的高度；第二，這個 sensor 是傾斜下視，不是平視前方。

具備這兩個特徵的軍方平台不多。MQ-9 Reaper 在 ISR 任務中常用的 MTS-B 球型 sensor、AC-130 系列的 MX-20 / MX-25 EO/IR、或者 RC-26 / U-28 等戰術 ISR 機的下視 pod，都會產生這樣的畫面。再對照 D 系列已揭露的觀測案件——伊拉克、敘利亞、約旦、UAE、阿曼灣這條中東 ISR 走廊——能看到地表的這片乾燥地貌，落在這片區域的可能性最高。

但官方什麼都沒說。

## 準星裡的那個黑塊

把視線拉回畫面中央。十字的正下方、距離準星中心只有一兩個刻度的位置，有一個明顯的暗色塊狀物。不像 A1 那個只有一兩個像素大的小黑點，A2 這個目標的形狀比較清楚一些——大約佔了畫面寬度的百分之一到二，輪廓微微圓潤，看起來像個立體物體的剪影。

當然，無法從這一張靜止畫面判斷它是什麼。

如果 sensor 設定在 Black Hot（黑色代表熱），這個黑塊代表它比沙漠地表更熱。沙漠白天可以熱到 60°C 以上，要比沙地還熱的物體，候選名單不多：噴射機的尾管、火箭、或者本身就有強熱源的東西。

如果是 White Hot（白色代表熱），這個黑塊就**比沙地還冷**。這正好對應到 D 系列累積到目前為止的三次「thermally cold UAP」紀錄：D23（UAE 上空 320/440 mph）、D38（波斯灣海面）、D44（亞丁灣海面）。每一次機組都在 IR 中看到一個比理論上「應該很熱」的背景還要冷的物體。

A2 是哪一邊？看不出來。HUD 上指示 polarity 的那塊資訊，剛好被覆蓋在頂部那條最寬的黑色長條底下。

## 那十二道黑條

那十二條 1.4(a) 黑色遮蔽不是隨機分布的。它們很整齊地切掉了 HUD 邊緣應該顯示資訊的位置——左上是任務日期時間、左中是 sensor pitch 角、左下是經緯度估算；右上是飛行高度與機速、右中是 sensor yaw、右下是機尾號或機組 ID；上方中段是 sensor 模式與 polarity；下方中段是 FOV 與 zoom 倍率。

換句話說，**畫面正中央的目標被留下來給你看，但是「在哪裡、什麼時候、怎麼看到的」全部被拿走**。

這是一個很有意識的取捨。從 AARO 的角度，目標的視覺呈現是可以公開的——民眾看得到、研究者可以分析形狀、影像分析師可以做幾何測量。但是「任務的位置、平台的能力、時間的關聯」是不能公開的，因為那會洩漏：
- 美軍在哪些區域常態執行 ISR
- sensor 的解析度與 FOV 等技術細節
- 對方（如果有的話）可能用來反推美軍偵察模式的時序資訊

所以畫面中央保留了一個未知物體的剪影，邊邊把所有的 context 都拿掉。

## 比 A1 更實在的證據

A1 與 A2 放在一起看，能感受到差別。A1 是「我們真的看到了某個東西，但你只能看到它在某個均勻背景前的小點」；A2 是「我們真的看到了某個東西，而且它和地面有明確的空間關係」。

對於後續分析來說，A2 是更實在的證據。地面細節提供了**比例尺**——可以推估目標距離 sensor 的相對遠近；提供了**運動參考**——如果有後續幀，可以判斷物體相對地面是否移動；提供了**環境約束**——這顯然不是高空氣球或衛星，是某個正在介入地表附近空域的東西。

「5」這個數字依然在準星右側。和 A1 一樣的位置、一樣的尺度。如果它代表 range gate，意味著兩張照片的 sensor 設定保持一致。如果代表 zoom level，意味著兩次截圖用的是同一個放大倍率。如果代表 target ID，那意味著 A1 與 A2 拍的可能是**同一個目標**在不同階段的兩個畫面。

最後這個可能性最有意思。FBI 把 A1 至 A8 放在一起公開，編號連續、命名規則一致、影像格式相同。這暗示它們屬於同一份檔案、可能是同一次觀測中擷取的多個瞬間。如果是這樣，A1 到 A8 構成一個**連續事件的關鍵幀序列**。

但這只是推測。在沒有時間戳、沒有任務代碼、沒有 sensor metadata 的情況下，剩下的只能是猜測。
