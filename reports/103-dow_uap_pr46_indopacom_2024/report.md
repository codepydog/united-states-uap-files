# #103 PR46 INDOPACOM 2024：9 秒 IR 影片，橄欖球狀主體 + 3 條徑向突出物

![cover](cover.jpg)

PR46 是 PR 系列中目標形態描述最具體的影片之一，地點記為 INDOPACOM（印太司令部 AOR），年份 2024。AARO 公開時無對應 D 系列 MISREP，但本片是 PR 系列首份標註明確「**結構特徵**」的影片：橄欖球狀（rugby/football-shaped）主體加上 3 條徑向突出物（1 條垂直、2 條向下 45 度）。

## 影片內容

- 長度：9.9 秒，1920×1080，30 fps
- 感測器：IR，HUD 邊角受 1.4(a) 黑塊遮蔽
- 對比區形態：
  - 中央為橄欖球狀（橢圓）暗區，沿水平軸延展
  - 上方 1 條垂直突出物
  - 下方左右各 1 條約 45 度斜伸的突出物
- 背景為均勻灰色（推測為高空雲層或開闊海面）

![中央橄欖球狀主體 + 3 條徑向突出物，影片中段](cover.jpg)

## 為什麼這份特別

PR46 是 PR 系列中對「結構」描述最具體的案件：

- 其他 PR 多為「contrast region」、「small thermal signature」等抽象描述
- PR46 caption 提到具體幾何「橄欖球 + 3 條突出物」，意味目標形態足夠清晰可被官方描述
- 突出物與主體的相對位置在 9 秒內保持穩定，排除「motion blur 殘影」可能

候選解釋包括：

- 旋翼直升機（橄欖球機身 + 旋翼下垂尾跡）：與「3 條突出物」拓撲不符
- 商業衛星發射殘餘（多級分離後的殘骸結構）
- 失控氣球（球體 + 多條繩索／配件）
- 真正未知物

AARO 列為 unresolved 即正面表示以上候選均不適配。

## 為什麼仍列為 unresolved

INDOPACOM 戰區涵蓋廣大太平洋，9 秒影片無 ground truth 與其他感測器交叉。形態獨特但 sample size = 1，無法做型態學分類。
