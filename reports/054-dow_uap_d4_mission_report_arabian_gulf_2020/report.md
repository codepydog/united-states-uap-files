## 為什麼 D4 是 D 系列遮蔽程度最高的案件

D4 PDF 共 5 頁，**前 4 頁完全遮蔽**（全頁僅顯示「1.4(a)」標記，意即依據 Executive Order 13526 Section 1.4(a) — 軍事計畫機密類別 — 整頁機密）。僅最後一頁殘留 GENTEXT/UAP 描述段。對比 D 系列其他案件：

| 案件 | PDF 頁數 | 完全遮蔽頁數 | 比例 |
|---|---|---|---|
| D3（2020 阿拉伯灣） | 7 頁 | 0 頁 | 0% |
| D10（2022 伊拉克） | 6 頁 | 0 頁 | 0% |
| D20（2023 敘利亞） | 9 頁 | 0 頁 | 0% |
| **D4（2020 阿拉伯灣）** | **5 頁** | **4 頁** | **80%** |
| D38（2020 中東） | 1 頁 | 0 頁 | 0%（但欄位填寫稀疏） |

D4 與 D3 同年同戰區（2020 阿拉伯灣 / USCENTCOM AOR），但兩份檔案的釋出策略完全不同：

- **D3** 是 USAF AFCENT MQ-9 案件 → 標準格式釋出（雖細節欄位遮蔽，但整體欄位框架可見）
- **D4** 是 PILOT（有人駕駛）案件 → 連任務報告框架都被遮蔽（4 頁全部 1.4(a)）

合理推斷：**D4 涉及有人駕駛任務的機型、單位、任務本身可能仍處於敏感狀態**，因此 USCENTCOM MDR 選擇將任務元資料整頁遮蔽，僅保留 UAP GENTEXT 段供 AARO 分析。

## 1. UAP 觀測核心內容

完整 PDF 殘存文字（除「1.4(a)」遮蔽標記）：

> GENTEXT/UAP
> • Description: (S//REL) AT 1258Z, [REDACTED] OBSERVED POSS UAP IVO 34SDG9041417044. BRIEF OBSERVATION PRECLUDED UAP ALTITUDE ESTIMATES. VELOCITY ESTIMATED AT 321 KNOTS. UAP INCREASED SPEED AND CHANGED DIRECTION TOWARDS THE EAST.
> PILOT: [REDACTED]
> [REDACTED]
> SECRET // REL TO USA, FVEY

> GENTEXT/UAP
> 描述：（機密 / 五眼可釋出）12:58Z [遮蔽] 於 MGRS 34SDG9041417044 附近觀測到 1 個可能 UAP。觀察時間短暫，無法估算 UAP 高度。速度估算為 321 KNOTS。UAP 加速並改變方向往東。
> 飛行員：[遮蔽]
> [遮蔽]
> 機密 / 五眼聯盟可釋出

**關鍵字「PILOT」** 出現在表單中（非 DGS aircrew、非 sensor operator）→ 這是**有人駕駛飛機**的觀測。D 系列以 MQ-9 / MQ-9-class UAV 觀測為主，D4 是少數明確標示「PILOT」的案件。

## 2. 「Possible UAP」hedge 語意

「**POSS UAP**」（possible UAP）是 D 系列中**較少見的 hedge 語意**。多數 D 系列案件直接寫「OBSERVED UAP」（觀測到 UAP）。「**possible**」一詞暗示飛行員不完全確定該物體是否歸類為 UAP，可能解讀：

1. **物體特徵不夠異常**：速度 321 KTS 在已知巡弋飛彈、無人機速度範圍內
2. **觀測時間太短**：BRIEF 字眼明確說明，無法充分判斷物體特性
3. **可能為已知飛行器**：飛行員當下難以排除為敵方飛彈／無人機／巡邏機

對應其他「possible」UAP 案件：[#055 D42](https://www.war.gov/UFO/#DOW-UAP-D42,%20Mission%20Report,%20Arabian%20Gulf,%202020) 與 [#053 D38](../053-dow_uap_d38_range_fouler_debrief_middle_east_may_2020/report.md)（D38 表格中也含 hedge 語意，雖未用「possible」字眼）。

## 3. 機動特徵：「加速 + 東轉」

「**INCREASED SPEED AND CHANGED DIRECTION TOWARDS THE EAST**」是 D 系列中常見的**反應性機動**描述。對比其他案件：

| 案件 | 機動描述 |
|---|---|
| **D4（2020 阿拉伯灣）** | **加速 + 東轉**（觀察期間機動） |
| D27（2024 Gulf of Oman） | 直線非機動 |
| D33（2023 東地中海） | Sharp 90 度直角轉彎 |
| D38（2020 波斯灣） | erratic movements above water |
| D23（2023 UAE） | ROTATE + 直線 |

「加速 + 東轉」可能對應：

- **反應性脫離**：UAP 察覺被觀測 → 加速逃離
- **預定軌道**：UAP 沿既定航跡執行，恰好於該時刻轉向
- **被風偏移**：UAP 為氣球類物體，被高空風帶往東方

321 KTS（≈ 595 km/h）為已知**亞音速軍機巡航速度**範圍（如 F-15E 巡航 ~Mach 0.85 = ~450 KTS, F-16 巡航 ~400 KTS, MQ-9 巡航 ~210 KTS）。**321 KTS 對 MQ-9 而言過快**，但對 F/A-18 / F-15 / F-16 而言為慢速巡航。

## 4. 地理矛盾：34S DG 對應地中海中部

座標「**34S DG 9041417044**」依 MGRS 規範：

- **34S grid zone**：經度 18°E–24°E（地中海中部，希臘南方海域）
- **DG square**：grid zone 內的 100 km 子方格

但檔案標題寫「Arabian Gulf」（阿拉伯灣，35°E–55°E）。**這存在 200+ km 的地理矛盾**。可能解讀：

1. **War.gov 標題錯誤**：對應 [#045 D27](../045-dow_uap_d27_mission_report_gulf_of_oman_june_2024/report.md)（標題 UAE Oct 2023 vs. 實際 Gulf of Oman 2024-06）、[#041 D20](../041-dow_uap_d20_mission_report_syria_essa_march_2023/report.md)（標題 Southern US 2020 vs. 實際 Syria 2023-03）等多份標題錯誤案件
2. **MGRS 解讀錯誤**：可能該座標應依其他 grid system 解讀
3. **跨戰區任務**：飛行員從地中海戰區跨入阿拉伯灣 AOR，觀測 grid 與報告框架戰區不一致

**最可能為標題錯誤**：D 系列 War.gov metadata 已知至少 4 處標題與 PDF 內容地理矛盾。D4 可能為第 5 處。**實際觀測地點可能在地中海中部，非阿拉伯灣**。

## 5. 觀察

**(1) D4 是 D 系列首份明確標示「PILOT」的有人駕駛案件**：D3 / D10 / D12 / D14 / D16 / D18 / D19 / D20 等案件皆為 MQ-9 DGS aircrew 觀測。D4 + D38 兩件 2020 年案件帶有明顯「有人駕駛戰機」signature，**可能來自 carrier air wing（CSG, USS Eisenhower / Truman / Reagan 等）的 F/A-18 機組**。對應 USCENTCOM 第五艦隊在波斯灣的固定部署。

**(2) SECRET // REL TO USA, FVEY 是 D 系列首見「可五眼釋出」案件**：對比 D32 / D27 / D25 的「SECRET // NOFORN」（不可外洩予外國），D4 的「REL TO USA, FVEY」（可釋出予美國 + 五眼聯盟）意味本案資料**允許英、加、澳、紐情報單位共享**。可能因 D4 觀測涉及多邊海軍演習或多國聯合反恐任務。

**(3) PDF 5 頁中 4 頁全 1.4(a) 遮蔽是 D 系列空前**：對比其他案件最多遮蔽 30-50% 欄位，但欄位框架仍可見。D4 連任務型態、單位、飛行員姓名、機種、起降基地都整頁遮蔽。**這支持「D4 屬於更敏感的活躍任務」假設**——可能涉及（仍進行中的）反伊朗或反恐情報任務。

**(4) UAP 速度估算「321 KTS」高度精確**：對比 D27（140 KTS）、D33（80 MPH）、D35（30 MPH）、D38（無速度估算）、D23（320/440 mph），D4 的「**321 KTS 精確值**」（非 ~320 / ~325 round number）意味機組可能用雷達或 sensor 鎖定取得精確 doppler 速度，**非目視估算**。但「觀測太短，無法估算高度」又表明觀測極短。可能解讀：機載雷達短暫鎖定取得速度但隨即失去。

**(5) D3 + D4 構成 2020 年阿拉伯灣 USCENTCOM「pre-AARO」連續觀測對**：兩案編號連續（D3, D4），都在 2020 年，地理座標雖矛盾但同戰區。對應 [#047 D3](../047-dow_uap_d3_mission_report_arabian_gulf_2020/report.md) 的「2020 年案件被 AARO 回溯整合進通報系統」推斷。D3 = MQ-9 編隊觀測, D4 = 有人駕駛單一觀測，**構成跨平台 2020 年 USCENTCOM 觀測對**。
