# #112 FBI Photo A6：這一張藏得最深

![photo](photo.png)

A1 到 A5 的遮蔽都還算節制。HUD 邊緣那十二條黑色長條覆蓋了 metadata，但畫面主體——sensor 看到的場景與目標——一直被保留。觀眾能讀到 sensor 紀錄的核心。

A6 不一樣。

打開檔案，左上角直接一大片連續的黑色，由多個重疊的方塊與長條組成，覆蓋了畫面**將近 1/4** 的面積。下方還有額外的黑條延伸進來。這種規模的遮蔽不只是遮文字，是**把整塊畫面區域整個拿掉**。

剩下能看的，是右上角一小塊地形（明亮的乾燥地表）、畫面中段的準星與目標、以及下方一條條被打斷的地表紋路。

## 為什麼這一張藏得這麼深

每一張靜止影像被官方公開前，都要經過 derivative classification review。審查員逐區看畫面，判斷哪些資訊**單獨被釋出時**仍會洩漏機密。可能的洩漏類型有三層：

第一層，**直接的 metadata**。日期、時間、座標、機體編號、aircrew 識別。這些都在 HUD 邊緣，用標準長條覆蓋就好。A1 至 A5 是這一層的處理結果。

第二層，**情境推導**。即使單一資訊不機密，組合起來可以推導出機密。例如：天氣型態 + 太陽方向 + 地形可以反推任務時間與位置；幾個目標的相對運動可以推算 sensor 平台的速度與航向；某種特殊的 HUD 圖示可以指出特定機型。如果 A6 的左上角原本含有這類「組合會洩漏」的元素，那就要用大面積遮蔽蓋掉。

第三層，**目標本身的某個特徵**。某些目標的影像如果直接公開，會洩漏美軍 sensor 的解析能力上限、特定演算法的追蹤模式，或讓對方（敵對情報機構）反推美軍能看到什麼、看不到什麼。

A6 的左上角那一大片遮蔽可能是第二層、第三層、或兩者混合的結果。原圖在那塊區域裡到底有什麼，現在只能猜。

## 剩下能看的部分

把左上角那塊遮蔽放一邊，看畫面其他部分。

準星位置在畫面中央偏右。中央交叉點的左下方，**那個熟悉的暗色圓塊狀目標**仍然存在。形狀、尺寸、與背景的對比度，全部與 A3 那個目標可以對應上。位置稍微偏離準星正中央。

地表紋路在右上角看得最清楚——亮色的沙質地，邊緣有少數較深的點狀痕跡（可能是低矮植被）。下方地形的紋路被多條 1.4(a) 黑條打斷，但能感受到還有起伏。整體與 A2、A3 的中東 ISR 走廊乾燥地貌呼應。

「5」依然在準星右側相同位置。連同前面五張，A1-A6 的 sensor 設定看起來沒換過。

## 連續性的暗示

如果 A1 到 A8 真的是同一場觀測的連續關鍵幀，那 A6 在這個序列裡的位置耐人尋味。前面 A3 是 boresight 對齊的明確版本、A4 是目標退遠的版本、A5 加上了第二個元素（小白圈）、A6 卻把更多東西藏起來。

為什麼？三種可能：

**第一**，A6 的拍攝瞬間，sensor 平台正在執行**特定模式切換**（例如從追蹤模式切到目標識別模式、或啟動 SAR 副雷達）。模式切換時 HUD 會顯示額外資訊，這些被遮掉。

**第二**，A6 的瞬間有**戰術情境資訊**出現在畫面上（例如其他友軍機的 datalink、敵對雷達警告、地對空威脅指示），這些不能公開。

**第三**，A6 本來就**不是 sensor 連續錄影中的一幀**，而是另一個 sensor 平台、另一個任務、另一次事件的影像，只是因為視覺風格相似被歸進同一個序列。

無法從這一張的內容判斷哪一種正確。但 A6 的遮蔽密度告訴我們一件事：**FBI 公開這些照片是經過嚴格篩選的，每一張的可見區域都是審查後的最大邊界**。
