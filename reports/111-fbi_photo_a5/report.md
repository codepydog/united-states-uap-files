# #111 FBI Photo A5：可能不只一個目標

![photo](photo.png)

到目前為止，A1 到 A4 都遵循同一個敘事節奏：**畫面中央一個目標，邊邊被遮蔽，「5」字在右側**。A5 打破了這個節奏。

仔細看畫面正上方，準星垂直延長線往上的位置。**有一個小白圈**。

不大，可能只佔畫面的 0.1%，但邊緣清晰，是個明確的圓形標記。位置就在頂部那條黑色遮蔽長條與準星之間的灰色背景上。它的形狀不像 sensor 雜訊（雜訊通常隨機散布、邊緣模糊），也不像背景反光（地面在那個距離不會產生小尺度高對比點）。比較像是**畫面中另一個被標註出來的東西**。

幾種解讀：

第一種，**第二個目標**。Sensor 操作員可能同時在追蹤兩個物體：一個放進準星正中央（畫面中央偏左下的暗色塊），另一個用標記符號標出位置（上方的小白圈）。對應 D 系列已揭露的多 UAP 案件——[D3](../047-dow_uap_d3_mission_report_arabian_gulf_2020/report.md) 在 27 秒內看到 4 個物體分 3 波，[D20](../041-dow_uap_d20_mission_report_syria_essa_march_2023/report.md) 在 FL600+ 看到 10-20 個亮物，[D42](../055-dow_uap_d42_range_fouler_debrief_japan_2023/report.md) 一度同畫面 3 個物體 moving amongst each other——多目標觀測不是稀有事件。

第二種，**位置標記符號（waypoint）**。某些 sensor 系統會讓操作員手動標出畫面中某個位置，方便後續分析。如果 A5 是後製分析階段加上的標記，這個白圈代表「**請注意這裡**」。

第三種，**HUD 元素未被完全遮蔽**。本應被 1.4(a) 蓋掉、但這個小白圈漏網了。如果是這樣，它可能是 sensor 系統內部的某個指示符號（如北方指示器、taxi/orbit 圖示、警告燈號）。

沒有時間戳、沒有 metadata，無法確認是哪一種。但它的存在本身就告訴我們：**A5 的畫面比 A1-A4 都多了一個元素**。

## 左上角那塊大黑方

A1 到 A4 的遮蔽都是規整的細長條，邊界整齊地切掉 HUD 邊緣的文字資訊。A5 不一樣。

左上角有一塊**大方形的黑色遮蔽**，邊長接近畫面寬度的 1/8。這比平常的 1.4(a) 長條大太多了，不像是要遮一行文字，比較像是要蓋掉**一整個資訊面板**。

軍方 sensor 顯示器常見的二級面板包括：
- **Moving map inset**：地圖小窗口，顯示 sensor 視點在地圖上的位置
- **Picture-in-picture（PIP）**：另一個 sensor 的副視野，例如主螢幕顯示 MX-20 的窄角，PIP 顯示同一目標的廣角
- **Range / Slant range display**：距離資訊面板
- **Target track summary**：追蹤目標的列表（一個畫面中可能多個 track）

如果這個大方塊原本是 PIP 副視野，那麼 A5 的原圖**含兩個 sensor 視角同步顯示**，與 [#046 D28](../046-dow_uap_d28_mission_report_ayn_al_asad_iraq_september_2024/report.md) 提到的 AC-130J MX-20 + MX-25 雙感測器配置相符。但官方把另一個視角蓋掉了。

A4 與 A5 的遮蔽手法不一樣：A4 用整片白色覆蓋左半邊，A5 用一個大黑方覆蓋左上角。但兩張的暗示相同——**原圖比公開的部分多含一個視角**。

## 主目標還在原位

把注意力拉回畫面中心。

準星正中央偏左下方，那個熟悉的暗色圓塊狀目標仍然存在。形狀、大小與 A3 接近，輪廓比 A4 銳利一些。位置稍微偏離準星正中央——大約在準星左下象限，距離交點約 1-2 個刻度。

如果 A5 是 A3 之後的某一個瞬間，這個位置漂移可能來自：
- 目標相對 sensor 平台移動
- Sensor 平台自身姿態變化（震動、轉向、抖動）
- 主動追蹤算法的延遲

地面紋路比 A3 多了一些細節，但整體仍是乾燥地表的調性。

「5」依然在右側準星附近的相同位置。設定保持一致。

## 從一個變兩個

如果上方那個小白圈真的是第二個目標，A5 在 D 系列已累積的多目標案件之外，又增加一個 FBI 公開影像維度的證據。前面那些案件都有完整的文字描述告訴你「**3 個物體同時在畫面上**」，但 A5 是**直接看得到**：兩個被標記的位置，主目標在準星中央偏下方，次目標在準星正上方。

這也部分解釋了為什麼 sensor 操作員選擇拍下這一張。在連續錄影中能擷取到「畫面同時呈現兩個被追蹤元素」的瞬間，本身就有特殊紀錄價值。

當然，這只是其中一個可能性。最保守的解讀是：白圈只是一個隨機沒被遮蔽的 HUD 圖示，意義平凡。但保守解讀沒辦法解釋為什麼 FBI 偏偏選了這張、把它放進公開的八張系列裡。
