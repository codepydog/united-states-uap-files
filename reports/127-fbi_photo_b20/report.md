# #127 FBI Photo B20：5 秒後目標還在原地

![photo](photo.jpg)

時間戳：**12/31/99 18:18:58**。距 [B19](../125-fbi_photo_b19/report.md) 5 秒。

B19 那個小暗點還在準星附近，B20 看下去，**位置幾乎沒變，但形態變了**。

## 「準星上方一點」→「兩個並排小塊」

B19（18:18:53）的目標是準星垂直線上方接近 0 mil 的位置，呈單純的小點。

B20（18:18:58）相同位置範圍，但**這次看起來像兩個並排的暗色小塊**，中間有個微小的灰色縫隙。比 B19 那種「**單點**」更接近後續 B22-B24（zoom in 之後）的「**雙塊**」形態。

意味著什麼？

如果 B19 的單點是真實情況，B20 的雙塊形態必須在 5 秒之內出現。要嘛是 sensor 解析度突然提升（不太可能，B20 仍是 ±15 mil 廣角），要嘛是物體本身**分裂或靠近**到能在這個 sensor 解析度下被分辨。

更可能的解讀是：**B19 的「單點」其實已經是兩個物體靠在一起**，只是太近以致 sensor 解析力把它們合成一個像素區。5 秒之後物體之間距離稍微拉開，於是在同樣的廣角畫面上能看出有兩個元素。

## 廣角下能看出雙塊很罕見

整個第一波觀測（B6 到 B5，2 分 20 秒）目標都呈現為**單一**的點或不規則團塊。從未在廣角 ±15 mil 模式下看到清楚的雙塊形態。

B20 是第一次在廣角下出現雙塊。意味著：

- 物體之間的角距離大到足以在 ±15 mil 解析度下分辨——對應目標距離 sensor **較近**或**橫向間距增大**
- 或 sensor 操作員剛把畫面整體優化，使對比與銳利度暫時增強

對應 [#120 B14](../120-fbi_photo_b14/report.md) 在 zoom in（±3 mil）下看到的明確雙塊形態——但 B20 是**廣角下也能看出來**，意味物體之間的相對距離至少有 5-10 mil 在這個視角下。
