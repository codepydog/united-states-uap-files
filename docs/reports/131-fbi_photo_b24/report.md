# #131 FBI Photo B24：複雜形態出現

![photo](photo.jpg)

時間戳：**12/31/99 18:19:40**。距 [B23](../130-fbi_photo_b23/report.md) 7 秒。

打開 B24，畫面中央偏左的位置，**一個看起來像「帶有翼狀延伸」的不規則暗色形狀**。比 B23 那個拉長雙塊更複雜：上方有突出尖端、下方有水平延伸、右側可能還有些隱約的細節。

整個 B 系列第二波觀測中，**B24 是形態最不規則的一張**。

## 「翼狀」是真的嗎

要小心解讀。

在 IR sensor 高放大下，**雜訊紋理也會被一起放大**。看起來像「翼」、「尾巴」、「凸起」的特徵，有可能是真實物體結構，也有可能是 sensor 像素雜訊與物體訊號疊加的視覺假象。

判別方法是看**形態在連續幀之間的一致性**。如果 B23（14 秒前）也呈現翼狀，B25 也呈現，那就是真實結構。如果只有 B24 這一張有翼狀、前後幀都沒有，比較可能是 sensor 偶然產生的假象。

B23 是拉長形態但沒有明確的翼狀。B13（14 秒後）是「主體 + 突起」也沒有翼狀。B24 的這個複雜形態，**在前後 14 秒的相鄰幀中無法被支持**。

但這也可能意味物體**真的會快速改變姿態**。某些 UAP 文獻描述的「**morphing**」現象就是這種「14 秒內形態大幅變化」的行為。

兩種解讀都成立。
