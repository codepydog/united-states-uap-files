# #080 DOW-UAP-PR20：科威特 2022-05，紅圈標註的單張靜止影像，長條對比區左上→右下漸亮

> **媒體來源**：PR20 在 War Department 2026-05-08 UAP release index 內列為 image-only 條目，但 war.gov UFO portal slideshow（公開白名單僅 PR19/26/34/35/38/43/45/46/49 共 9 條）、DVIDS、以及 Wayback Machine 自 2026-05-08 至今 58 個 snapshot 均查無對應 image asset。以下描述依官方 caption 與對應 [#036 D12](../036-dow_uap_d12_mission_report_iraq_may_2022/report.md) MISREP 推得；本條目作為 placeholder 保留，待未來補釋出再補圖。

PR20 是 PR 系列中唯一以「單張靜止影像」釋出的條目，其他 PR 多為短影片片段。依官方 caption，畫面以紅圈標註物體位置，紅外背景上一個長條形對比區呈現左上至右下的亮度梯度，無法判斷是運動軌跡的瞬間還是物體本身形狀。

## 影像內容

單張靜止影像，紅圈手動標註。長條對比區位於畫面中央偏上，左上端較暗、右下端較亮，可能是物體姿態造成的 IR 對比差，也可能是 sensor 殘影。沒有影片給出時間序列，無法計算速度或軌跡。

## 對應 D 系列 MISREP

對應 [#036 DOW-UAP-D12](../036-dow_uap_d12_mission_report_iraq_may_2022/report.md)（伊拉克 2022-05-20，OP PHANTOM FLEX，196 ATKS MQ-9 在 18,000 ft 觀測 1 個 UAP，20:43Z）。D12 的 MISREP 描述提到 1 個 UAP 短暫通過視野，PR20 的靜止影像很可能是該觀測時刻的單一截幀。

PR20 名稱含「Kuwait」是因為 196 ATKS MQ-9 的起降基地是 OKAS（Ali Al Salem AB, Kuwait），但實際任務區在伊拉克。這個命名差是 PR 系列釋出時依「起降基地」標籤而非「任務區」標籤。

## 為什麼這份未解

- 單一截幀，無時間序列可推估運動學
- 紅圈手動標註本身代表「無自動 track lock」，sensor 沒有持續追蹤
- 對比區形狀（長條）可能是 motion blur，也可能是物體姿態，無法以單幀區分
