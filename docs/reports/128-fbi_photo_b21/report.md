# #128 FBI Photo B21：8 秒之內維持雙塊

![photo](photo.jpg)

時間戳：**12/31/99 18:19:06**。距 [B20](../127-fbi_photo_b20/report.md) 8 秒。

B20 的雙塊形態到 B21 還在。位置幾乎沒變、形態幾乎沒變、廣角設定一樣。**雙塊形態維持至少 8 秒**，意味這不是 sensor 雜訊偶然產生的視覺假象，是物體真實的呈現。

## 形態穩定，sensor 沒切換

B19 → B20 → B21 的這 13 秒，sensor 一直在廣角模式（±15 mil），目標位置幾乎不變。如果是兩個物體並排飛行，**它們的相對位置與運動向量幾乎完全同步**。

下一秒（35 秒後的 [B22](../129-fbi_photo_b22/report.md)），sensor 才切換到 zoom in（±3 mil）。從 B21 到 B22 中間 13 秒可能就是**操作員執行 zoom 動作的時間**：拉廣，確認對齊，拉近，重新校準。
